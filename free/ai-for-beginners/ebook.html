The basics

What is AI, really?

Forget the sci-fi robots. The AI you're going to use is simpler than that: it's a brilliant assistant you talk to by typing.

Tools like ChatGPT, Claude, and Gemini are called chatbots or AI assistants. You type a request — a question, a task, an idea — and it writes back an answer in seconds. It can draft an email, explain a topic, brainstorm names, write captions, plan a trip, summarize a document, or help you code. You "chat" with it like you'd text a very fast, very well-read friend.

It reads & writes

Emails, captions, scripts, summaries, ideas — text in, text out.

It explains

Ask it to break down anything "like I'm 10." It's a patient tutor.

It creates

Some tools also make images, analyze photos, and read files you upload.

One thing to remember

AI is confident — even when it's wrong. It can "make things up" (you'll hear this called a hallucination). So for facts that matter — names, dates, numbers, medical or legal info — always double-check. Treat it like a sharp intern: incredibly helpful, fast, and worth a second look.

The mindset shift: you don't need to "know everything." You need to know how to ask. That's a skill — and it's learnable in an afternoon.

Talk the talk

The words, decoded

You'll see these everywhere. Here's what each one actually means — no PhD required.

Prompt
What you type to the AI. Your request, question, or instruction.
Model
The "brain" doing the work (e.g. GPT-5, Claude Sonnet, Gemini). Newer = smarter, usually.
Token
How AI counts text — roughly ¾ of a word. Mostly matters for length limits and pricing.
Context window
How much it can "hold in mind" at once. Bigger = you can paste longer documents.
Hallucination
When AI states something false with total confidence. Always verify facts.
System prompt
Hidden instructions that set the AI's behavior. You usually don't see it.
Multimodal
Can handle more than text — images, audio, files. Most modern tools are.
Chat / Thread
One ongoing conversation. It remembers everything within that chat.
Fine-tuning
Training a model on your own data. Advanced — skip it for now.
API
A way for apps to plug into the AI with code. For builders, not beginners.
Generative AI
AI that creates new things — text, images, video — rather than just sorting data.
Prompt engineering
The skill of writing prompts that get great results. You're learning it right now.

You don't need to memorize these. Bookmark this page and glance back. Within a week they'll feel like second nature.

The real skill

How to write a prompt that works

Bad prompt in, bad answer out. The fix is a simple formula I use every day. Four ingredients:

1

Role

Tell it who to be. "You are a friendly fitness coach…"

2

Task

Say exactly what you want. "…write a 7-day beginner plan."

3

Context

Give the details. "I'm 35, busy, no gym, 20 min a day."

4

Format

Shape the output. "As a simple table I can print."

See the difference

❌ Weak: "Give me a workout."
✅ Strong: "You are a friendly fitness coach. Write a 7-day beginner workout plan for a busy 35-year-old with no gym and only 20 minutes a day. Give it as a simple day-by-day table, and keep it encouraging."

Same tool. Wildly different result. The strong prompt got a role, a clear task, real context, and a format. Specific beats clever every time.

Golden rule: if the answer isn't what you wanted, don't start over — just reply and refine: "Make it shorter," "More casual," "Add a grocery list." It's a conversation.

Steal these

7 copy-paste starter prompts

Open any of the three tools, paste one in, and fill in the blanks. Instant momentum.

WRITE A POST
Write 3 short Instagram captions for [my business] about [topic]. Friendly tone, 1 emoji each, include a call to action.
EXPLAIN ANYTHING
Explain [confusing topic] to me like I'm 12, using a simple real-life example. Then give me 3 things most beginners get wrong.
SAVE TIME ON EMAIL
Write a polite, professional reply to this email saying [your point]. Keep it under 120 words. Here's the email: [paste it].
PLAN YOUR WEEK
I have these tasks: [list them]. Build me a realistic daily plan for the week as a table, with the hardest task first each day.
BRAINSTORM NAMES
Give me 15 catchy name ideas for [thing]. Mix short/punchy and creative. Add a one-line reason for each.
LEARN A SKILL
Act as my tutor for [skill]. Make me a 5-step beginner roadmap, and ask me 1 question to check I understood each step.
FIX MY WRITING
Improve this so it's clearer and more confident, but keep my voice. Don't make it stiff: [paste your text].
